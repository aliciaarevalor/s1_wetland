## Using Sentinel-1 imagery for water cover detection in a region with high surface water variability

This repository aims to reproduce the exercise https://github.com/webandras/gee_s1_sar_wetland with the purpose of having a better understanding of the use of SAR data, especially Sentinel-1 imagery, and its application on water cover detection.
Code in Google Earth Engine: https://code.earthengine.google.com/?accept_repo=users/alarevalo/s1_wetlands
Original Google Earth Engine repository: https://code.earthengine.google.com/?accept_repo=users/gulandras90/inlandExcessWater
